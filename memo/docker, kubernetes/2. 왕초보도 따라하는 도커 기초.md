# 왕초보도 따라하는 도커 기초

## 도커 설치하기

```
# sudo -i //root 권한 획득
# apt install docker.io //docker 설치
# docker //도커 도움말
```

```
//서비스 올리기 
#docker search tomcat //이미지 검색
# docker run -d -p 8080:8080 --name [이름] [검색에나온명] 
```

* -d : 백그라운드 실행
* -p : 포트 지정
* --name : 이름 지정

```
//톰캣 설치
# docker run -d -p 8080:8080 --name tc consol/tomcat-7.0
```
