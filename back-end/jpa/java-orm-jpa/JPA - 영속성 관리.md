# 영속성 관리

## 영속성 컨텍스트

* 엔티티를 영구 저장하는 환경
* 영속성 컨텍스트는 논리적인 개념이다.
* 엔티티 매니저를 통해서 영속성 컨텍스트에 접근.



## 엔티티의 생명주기

* 비영속
  * 영속성 컨텍스트와 전혀 관계가 없는 새로운 상태
  * 객체를 생성하였지만 EntityManager와 아무런 관련이 없는 상태
* 영속
  * 영속성 컨텍스트에 관리되는 상태
  * 생성한 엔티티 객체를 영속성 컨텍스트 안에 집어 넣은 상태
* 준영속
  * 영속성 컨텍스트에 저장되었다가 분리된 상태
* 삭제
  * 삭제된 상태



## 영속성 컨텍스트의 이점

* 1차 캐시
* 동일성 보장
* 트랜잭션을 지원하는 쓰기 지연
* 변경 감지
* 지연 로딩
