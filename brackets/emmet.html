TAG 관련 Emmet 단축키

  • ! + tab = html 문서 생성
  • ul, li 클릭후 tab 클릭 -> 태그 자동 생성
  • ul>li*5 -> ul 에 li 가 5개 들어있는 태그 생성
  • .list-item + tab -> list-item class 가진 div 생성 #list-item + tab -> list-item id를 가진 div 생성 .list-item.new-product -> 클래스 2개가진 div 생성 .list-item#new-product -> list-item class 를 가지고 new-product id 를 가진 div 생성

Style 관련 Emmet 주요 단축키

  • c + tab -> color
  • bgc + tab -> background-color
  • f + tab -> font
  • p20 + tab -> padding : 20px;
  • m20% + tab -> margin : 20%;
  • m20p + tab -> margin : 20%;
  • fz2em + tab -> font-size : 2em;
  • tac + tab -> text-align : center;
  • tal 의 경우 text-align-last 가 된다.
  • ta:l + tab 과 같은 형식이 정석이다.
  • pos:r -> position:relative ...
  • lh30 -> px가 붙지 않는다. -> lh30px 로 해야한다.
  • 기타 참고
