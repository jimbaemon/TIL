  • Ctrl + Shift + D : 한줄 제거

  • Ctrl + Shift + / : 범위 주석 (토글키)
  • Ctrl + Alt + H : 사이드바 열기 닫기 (토글키)
  • Ctrl + Alt + H : 사이드바 열기 닫기 (토글키)
  • Ctrl + Alt + ↑,↓ : 다중 입력, 다중 수정키1
  • Ctrl + 마우스왼쪽 : 다중 입력, 다중 수정키2
